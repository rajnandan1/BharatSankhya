Sankhya

"Sankhya" is a Browser extension designed to provide users with a quick and convenient way to convert numerical figures into their corresponding Indian word representations. This extension is a useful tool for individuals who need to work with numerical data, such as currency amounts or numerical values in reports, documents, or web content. It simplifies the process of transforming numbers into words, which is a common requirement in various contexts.

Features

  • Works by text selections
  • Parses separators like comma. Example: 1,005 to One thousand five
  • Color gradient based on size of the number
  • Max limit 999999999999999
